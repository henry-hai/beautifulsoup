# BeautifulSoup Extended

This repository contains an extended version of BeautifulSoup with additional APIs for Milestone 2.

## Installation

```bash
pip install beautifulsoup4
```

## Repository Setup

This repository contains:
- `bs4/` - Core BeautifulSoup library with extensions
- `apps/m2/` - Milestone 2 application programs
- `bs4/tests/` - Test cases for new APIs

## Milestone 2 Features

- **SoupStrainer** - Optimized parsing for large files
- **SoupReplacer** - Tag replacement during parsing

## Usage

See individual application files in `apps/m2/` for examples.
